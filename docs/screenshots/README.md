# Screenshots Directory

This directory is used to store all screenshots and visual assets for the CareerUpdates documentation.

## Required Assets

To complete the visual documentation of the project, please ensure the following screenshots are captured and placed in this directory:

| Filename | Description | Dimensions (Recommended) |
|----------|-------------|--------------------------|
| `homepage.png` | The main landing page showing the search and recent listings. | 1920x1080 |
| `job-details.png` | A single job listing showing the AI summary and full markdown description. | 1920x1080 |
| `admin-dashboard.png` | The primary admin view showing overall metrics. | 1920x1080 |
| `new-job.png` | The URL import interface where the AI extraction is triggered. | 1920x1080 |
| `database-audit.png` | The quality control screen showing flagged or broken jobs. | 1920x1080 |
| `ai-settings.png` | The configuration panel for OpenAI prompts and extraction rules. | 1920x1080 |

## Best Practices for Screenshots

- **Consistency:** Use the same browser window size for all desktop screenshots.
- **Cleanliness:** Hide bookmarks bars, extensions, and personal information.
- **Theme:** Use the same theme (Dark/Light) across all screenshots for a unified look.
- **Optimization:** Compress images using a tool like TinyPNG before committing to reduce repository size.
